Project Title: MiniGenAI
Problem Statement: Many beginners use AI tools without understanding how generative models work internally.
Methodology: A decoder-only Transformer was built from scratch using PyTorch and trained with next-character prediction.

Conclusion: MiniGenAI demonstrates the foundation of GPT-style generative AI.
